reports, and superseded/deprecated designs. They are preserved here for historical
reference but are **no longer authoritative**.

## Why these were archived

- **`*_FIX*.md` / `*_FIX_SUMMARY.md`** — one-off bug-fix write-ups from past sessions.
- **`*_COMPLETE.md` / `*_SUCCESS.md` / `*_VICTORY.md`** — milestone completion reports;
  their content now lives in [../CHANGELOG.md](../CHANGELOG.md) and
  [../MODULE_STATUS.md](../MODULE_STATUS.md).
- **`*_IMPLEMENTATION*.md` / `*_SUMMARY.md`** — implementation logs superseded by
  [../ARCHITECTURE.md](../ARCHITECTURE.md).
- **`THE_FRIGATE.md` / `NO_MORE_IFRAMES.md` / `NAVIGATION_*` / `ARCHITECTURE_FIX_NO_IFRAMES.md`**
  — deprecated navigation/iframe designs; current model is
  [../SIMPLE_NAVIGATION.md](../SIMPLE_NAVIGATION.md) and
  [../WEB_UI_ARCHITECTURE_DECISION.md](../WEB_UI_ARCHITECTURE_DECISION.md).
